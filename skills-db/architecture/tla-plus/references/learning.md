# Learning Path

## Fastest primary route

1. Lamport video course for the conceptual on-ramp.
2. PlusCal tutorial if you learn best from algorithms and immediate checking.
3. `Specifying Systems` for the canonical reference and examples.
4. `tlaplus/Examples` when you want validated case studies instead of isolated snippets.

## Good first practical loop

- Install VS Code extension or set up the CLI.
- Start with a very small PlusCal or TLA+ model.
- Run TLC on a tiny configuration.
- Read the counterexample before making the model larger.

## Good written sources

- `Specifying Systems` for language and TLC foundations
- Lamport high-level view page for modeling posture
- Official wiki learning page for a compact entry path
- Hyperbook for broader tutorial material

## When to branch out

- Learn PlusCal after basic TLA+ if you need algorithmic syntax.
- Read the examples corpus when you want realistic patterns.
- Read TLAPS material only after the safety model has stabilized.
- Read Apalache material only if symbolic checking or bounded SMT-based exploration is actually useful.
