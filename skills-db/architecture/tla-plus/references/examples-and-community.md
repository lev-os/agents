# Examples and Community

## Official examples corpus

`tlaplus/Examples` is useful for three different jobs:
- learning patterns from validated specs
- finding examples of specific features
- testing tools against a diverse real corpus

## How to mine the corpus

Search `manifest.json` files for:
- `pluscal`
- `proof`
- `symbolic`
- `deadlock failure`
- `liveness failure`
- `assumption failure`

Search `.cfg` files for:
- `SYMMETRY`
- `VIEW`
- `ALIAS`
- `CONSTRAINT`
- `DEADLOCK`

## Good starting pockets

- `specifications/SpecifyingSystems/` for book-aligned material
- beginner-friendly examples listed in the repo README
- protocol-heavy examples when you need realistic distributed-system patterns

## Community paths

- Google Group: `https://groups.google.com/forum/#!forum/tlaplus`
- Subreddit: `https://www.reddit.com/r/tlaplus/`
- Stack Overflow tag: `https://stackoverflow.com/questions/tagged/tla%2b`

## Extra ecosystem surface

- `tlaplus/CommunityModules` for reusable modules outside the built-in standard modules
- `tlaplus/vscode-tlaplus` knowledge base for pragmatic workflow fixes and common mistakes
