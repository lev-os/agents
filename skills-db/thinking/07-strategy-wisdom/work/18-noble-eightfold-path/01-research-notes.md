# Research Notes: Noble Eightfold Path

## Origin
- **Author**: Buddha (Siddhartha Gautama)
- **Year**: ~500 BCE
- **Source**: Buddhist Canon (Pali Canon, Tripitaka)

## Core Concept
The Noble Eightfold Path is Buddhism's practical framework for liberation from suffering. It consists of eight interconnected practices grouped into wisdom (right view, right intention), ethics (right speech, right action, right livelihood), and mental discipline (right effort, right mindfulness, right concentration). Unlike a linear checklist, these practices work holistically—progress in one area supports progress in others.
