# Work Command Matrix (Process-Only)

Use this file for `/work` process routing terms.

## Primitives

| Primitive | Aliases | Intent |
|---|---|---|
| `get` | `find`, `search`, `lookup`, `read`, `scan` | Gather context/evidence |
| `work` | `plan`, `design`, `spec`, `implement`, `execute` | Run lifecycle loop |
| `ask` | `wiz`, `wizard`, `guide me`, `unstuck` | Clarify underspecified intent |
| `check` | `validate`, `align`, `scan` | Validate readiness/quality |
| `handoff` | `checkpoint`, `resume`, `exit` | Update session continuity |

## Operational Expectations

1. Start with handoff + entity matrix.
2. Append ticks on substantive turns.
3. Validate before close.
