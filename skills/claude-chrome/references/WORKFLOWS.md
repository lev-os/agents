## Development Testing

### Test Local Development

```
Open localhost:3000, test the new form validation, check console for errors,
and screenshot any issues you find
```

### Console Debugging

```
Open the dashboard page and check the console for any errors when
the page loads
```

### Cross-Browser Testing

```
Open the signup flow in different viewport sizes (mobile, tablet, desktop)
and screenshot each. Note any layout issues.
```

### API Response Validation

```
Open the Network tab, trigger the user profile API call,
and verify the response structure matches our schema
```
