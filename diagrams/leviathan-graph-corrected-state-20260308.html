1 — Plain English
The Research Project

A corpus analysis POC

The project is a collection of prompts from major IDEs and agent tools plus a deterministic analysis layer you added on top.

  • Take raw prompts and tools
  • Extract rules, atoms, and structures
  • Build graph views
  • Compare convergence and differentiation
FlowMind

The conceptual engine

FlowMind matters because it is designed around structured declarations and can plausibly do prompt-to-rule and rule-to-prompt transformations better than a one-off script pile.

  • Control/policy plane
  • Good fit for declarative classification
  • Not yet fully wired in as the graph brain
Leviathan Graph

The native runtime direction

core/graph is the beginning of a native graph runtime for Lev. The end goal was to dogfood until it could recreate the prompt-analysis loop now done in Python.

  • Event to entity mutation path is real
  • Specs are larger than implementation
  • Native parity is not done yet

4 — What Nanograph Is And What It Did
Nanograph

A local typed graph database

Nanograph is not the Leviathan graph. It is an embedded graph database with:

  • .pg schema files
  • .gq query files
  • .jsonl seed/load data
  • .nano local DB folders
What We Proved

Only the export seam

We did not adopt Nanograph into core/graph. We proved one narrow thing:

  • The Leviathan harness compiler can emit a Nanograph-compatible pack.
  • That pack can be init'd, load'd, check'd, and run'd.
  • So Nanograph is a viable compiler-side experiment target.
This is “Nanograph as export target,” not “Nanograph solved Lev graph.”

6 — What The Research Actually Proved
Proved
  • The prompt-corpus analysis pipeline works as an external Python system.
  • The corpus is highly convergent under the current extractor.
  • Leviathan can analyze its own harness graph with a similar deterministic pipeline.
  • Leviathan can export that harness graph into Nanograph.
Not Proved
  • That PromptOS is a product or moat.
  • That Nanograph should be Lev’s graph backend.
  • That Leviathan natively reproduces the full prompt-analysis loop yet.
  • That current cross-corpus density comparisons are extractor-robust.
