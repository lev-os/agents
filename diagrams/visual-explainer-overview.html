Anti-Slop System
Before — Generic AI Output
  • Inter font on everything
  • Purple gradient text on headings
  • Emoji icons in every section header
  • Glowing card shadows with animated pulse
  • Cyan + magenta + pink neon palette
  • Every card identical — no visual hierarchy
  • Three-dot window chrome on code blocks
After — visual-explainer Output
  • Distinctive font pairing per page (13 options)
  • Constrained palettes — earthy, editorial, precise
  • Colored dot labels, not emoji
  • Depth tiers: hero, elevated, default, recessed
  • 6 curated aesthetic directions with guard rails
  • Asymmetric layouts with intentional visual weight
  • Simple file header on code blocks — no cliches
New in v0.7.0

Print Stylesheet

Cmd+P produces a clean, light-background PDF. Hides interactive controls, linearizes slides, force-opens collapsible sections.

Theme Toggle

In-page light/dark switching without changing OS settings. Three-state cycle: OS default, opposite, back to OS.

Prism.js Syntax Highlighting

Code blocks with keyword coloring matched to the page palette. Autoloader fetches language grammars on demand.

Code Diff Panels

Line-level before/after comparison with line numbers, green/red highlighting, and file headers.

Touch Pinch-to-Zoom

Two-finger zoom and pan on mobile for Mermaid diagrams. Prevents browser native pinch from interfering.

Comparison Diagram Type

Structured tradeoff analysis: hero summary, candidate cards, dimension-by-dimension table, deep dives, recommendation.

Unified Command Interface

Single entry point with arg-based routing. /visual-explainer <type> [args] [all|publish] — no separate slash commands needed.

Tiered Quality Gates

Three modes: default (local), all (responsive + print + slop + fact-check), publish (all + here.now deploy). External audiences get maximum scrutiny.

Decision / Feedback Capture

Interactive decision pages with localStorage auto-save. Radio options, notes, progress tracking, JSON export. Reload-safe — your choices persist.

Run with Claude

CLI runner button on decision pages. Builds a prompt from your choices, copies claude -p command to clipboard. Paste and go.

here.now Integration
visual-explainer + here.now = instant shareable diagrams. Generate any diagram, then publish it to a live URL with a single command. No deploy pipeline, no build step — just ./scripts/publish.sh and share the link.
