Architectural Invariants

✓

Domain is a leaf

Zero upstream dependencies. Types can never create import cycles.

✓

FlowMind ≠ Orchestration

Definition has no dependency on execution. Clean separation of concerns.

✓

Correct dependency direction

Execution depends on definition. Never the reverse. Orchestration → FlowMind → Domain.

✗

No function re-exports

FlowMind re-exports types (free), never functions from orchestration. Prevents cycle.
