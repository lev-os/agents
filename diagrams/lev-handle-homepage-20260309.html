The core model

Leviathan treats agents as systems, not chatbots. The core model is simple: FlowMind decides, Graph remembers, and Event Bus records. That split matters because it keeps policy, state, and causality explicit instead of collapsing everything into a single black box.

FlowMind

Control

The declarative control and policy substrate. Routing, constraints, and executable declarations live here.

Graph

State

The state and knowledge plane. Entities, claims, evidence, traversal, and projections live here.

Event Bus

Causality

The canonical spine. Everything important emits events, making the system replayable and auditable.
