matrix-widget / drop-in parity heatmap

Standalone HTML + CSS + JS component. One .matrix-widget element, one data-matrix JSON payload, auto-hydrates on DOMContentLoaded. Four cell states (direct/family/adjacent/none), row+col hover highlight, binding panel, chip lane filter, sticky first column, mobile horizontal scroll.

Contract
<link rel="stylesheet" href="matrix-widget.css">
<div class="matrix-widget" data-matrix='{"rows":[...],"cols":[...],"cells":{...},"laneFilters":[...]}'></div>
<script type="module" src="matrix-widget.js"></script>
