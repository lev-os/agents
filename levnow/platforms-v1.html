PRINCIPLES

Five load-bearing invariants

1

. Pentagon-only auto client set

claude-code, codex, opencode, pi run in auto lanes. cursor is manual/quarterly. VSCode explicitly out of scope.

2

. Single source of truth per concern

DNA canon amended in-place. No forked overlay definitions. No path-local exceptions (AGENTS.md §2.3).

3

. Contracts as executable documents

Capability manifests and parity contracts are enforced by registered DNA gates — not narrative docs.

4

. Flat workstream shape

.lev/pm/workstreams/<id>/state/workstream.yaml + captures/. 25/25 precedent preserved. No nested children.

5

. Cursor is manual-verified

Quarterly smoke refreshes last_verified_at. NOT in nightly pentagon-live. Per-tier staleness (30d auto / 90d cursor).

DECISION DRIVERS

What ranks the options

DD1 — Canon enforces promises

No orphan gates. Every plan-declared gate resolves in dna/gates.yaml before enforcement runs.

DD2 — Nightly cost bounded

Pentagon-live lane stays within p99 < 50ms per manifest write path.

DD3 — Unambiguous ownership

Executor never guesses filesystem location. One module owns each deliverable (AGENTS.md §2.2).

PRE-MORTEM

Four scenarios with mitigations

Each scenario has failure · detection · mitigation · owner · threshold.

1. DNA merge stampede

Failure: 200+ overlays at load blow p99. Detection: dna_merge_duration_ms p99 > 50ms. Mitigation: static merge only; Option B revisit trigger (3 nightly breaches). Owner: core/config. Threshold: 3 breaches → file platforms-v2-dna-live-overlay.

2. Cursor staleness false positives — RESOLVED

Was: iter-2 single 30d SLO → cursor red-flagged 3×/year. Fix: per-tier map (auto 30d, manual 90d). Owner: plugins/platforms. Threshold: eliminated.

3. Manifest drift

Failure: adapter I/O shape changes, manifest doesn't. Detection: capability-manifest-schema gate fails pre-commit. Mitigation: nightly pentagon-live refreshes last_verified_at; 14d warn / 30d red. Owner: plugins/platforms.

4. pi model-vs-client confusion (NEW)

Failure: contributors assume pi = model only, skip pentagon harness enrollment. Detection: D0.5 pentagon_clients.pi absent → unit gate red. Mitigation: Principle 5 names adapter path; D0.6 seeds fixture lane. Owner: plugins/platforms. Threshold: 1 mismatch → CI red + needs-adr label.
