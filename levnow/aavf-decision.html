Evidence

10 facts now empirically established

Each cite-able to file:line. Each survives adversarial probing.

1. B3 hybrid IR is justified

Only composition that honestly passes all 3 cross-IR patterns (triple, overlay, mixed). B2 fails 2/3. B1 fails triple+overlay. Hybrid is not redundant sugar.

2. B1 reification 2x tax

TPM 2.80 hops vs REF 1.40 hops on standard query mix. Triple-only forces ~2x lookups for getEntity/lineage compared to hybrid IR's overlay shortcut.

3. B1 closed RDF vocab rejects

11 hardcoded predicates. Unknown predicate → reject at op-level. Cannot extend vocabulary at runtime without schema migration.

4. C5 locality default-on works

Reference substrate coerces missing scope to 'global'. Round-1 finding fixed. Locality enforces by default, not opt-in.

5. Permissiveness ≠ correctness

Composition gate measures whether substrate accepts ops. Attacker passes 3/3 by accepting everything. WINNER must pass capability matrix honestly.

6. Strict falsification matcher

outcome === attack.expected required. Empty ops → INVALID, not vacuous PASS. Round-1 BLOCKER #1 closed.

7. add_link wiki bug fixed

InMemoryGraphAdapter.apply() now handles add_link/add_claim/update_claim/remove_link. Wiki: 0 → 8 link bullets materialized.

8. Sqlite index = 60x P99

Compound index (source_id, target_id) on links. Q P99: 42.87ms → 0.72ms. Cost: +1.6 MB storage, +12% ingest.

9. Real LLM compositor latency

claude-haiku-4.5 via OpenRouter. 50 calls × 4 candidates. P50 781ms / P99 1782ms. Within 3000ms threshold. Cost ~$0.005/battery.

10. B3 doublewrite cost = 3x

Every claim records as claim AND overlay AND tick. Amplification 3.00 measured. Acceptable for read-heavy workloads.

What if this all ran in the background?
