statline — reusable KPI row

Component · 4 demos · theme-agnostic (CSS custom props only)

01 · Default 4-cell (GXW parity stats)

Original GXW variant masthead — 13 impls, 12 flows, 8 lanes, 5 outliers. Responsive 4→2x2→1.

<div class="statline" data-stats='[
  {"k":"Graph impls","v":"13","unit":"simulated"},
  {"k":"Projection flows","v":"12","unit":"minimum set"},
  {"k":"Promotion lanes","v":"8","unit":"ordered"},
  {"k":"Uniqueness outliers","v":"5","unit":"high-leverage"}
]'></div>

02 · Financial (string values, 2-col variant)

Mixed-format values (currency, percentages, index). Uses .statline--2col for always-2 layout.

<div class="statline statline--2col" data-stats='[
  {"k":"Revenue (TTM)","v":"$4.2M","unit":"+18% YoY"},
  {"k":"MRR","v":"$380K","unit":"as of Apr"},
  {"k":"Churn","v":"2.1%","unit":"monthly"},
  {"k":"NPS","v":"62","unit":"detractor-adj."}
]'></div>

03 · DevOps (integer values, animated on scroll)

Integer cells tween 0 → target once (800ms). Scroll this block into view to trigger.

↓ scroll down to trigger counter animation ↓
<div class="statline" data-animate="true" data-stats='[
  {"k":"Uptime","v":"99","unit":"% p99"},
  {"k":"P95 latency","v":"142","unit":"ms"},
  {"k":"Deploys / wk","v":"47","unit":"green"},
  {"k":"Incidents","v":"3","unit":"this qtr"}
]'></div>

04 · Inline mini-stat (pipe-separated)

Compact single-line form for bylines, footers, table captions. Wraps gracefully on narrow viewports.

↳ same component under Daybreak light palette
<div class="statline statline--inline" data-stats='[
  {"k":"Impls","v":"13"},
  {"k":"Flows","v":"12"},
  {"k":"Lanes","v":"8"},
  {"k":"Outliers","v":"5"},
  {"k":"Rev","v":"0.79"}
]'></div>
